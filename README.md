# 🧩 Mastermind Digital Logic Implementation

This project was developed for the **Digital Systems** course during my first year of Computer Engineering. The goal was to digitally reconstruct the classic **Mastermind** board game logic using basic hardware components.

## 📝 About the Project
The implementation was built using the **Digital** simulator, focusing on low-level hardware design rather than traditional software coding.

### ⚠️ Status: Collaborative Debugging
As an aspiring engineer, I value transparency and peer review:
* **Academic Success:** The project earned a good grade.
* **Known Bug:** There is a logic error in the **Finite State Machine (FSM)**.
* **Call for Action:** If you are a digital logic enthusiast, feel free to **Fork** this repository and submit a **Pull Request** with a fix for the state machine logic.

## 🛠️ Concepts Explored
* **Boolean Logic:** AND, OR, NOT, and XOR gates.
* **Sequential Logic:** Registers, Flip-flops, and State Machines.
* **Component Design:** Adders, Comparators, and Multiplexers.
